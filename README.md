# TurnNoon

창업하기 전, 실패할 가능성부터 확인하세요.

## 소개

TurnNoon은 예비 창업자가 큰돈을 투자하기 전에 손익분기점·폐업 이력·상권 위험을 데이터로 진단하는 **창업 리스크 진단 플랫폼**이에요.

기존 창업 서비스가 "좋은 상권"을 추천하는 데 집중한다면, TurnNoon은 위험한 조건을 발견하면 **"이 조건은 위험합니다"** 라고 알려드려요.

## 핵심 기능

- **손익분기점 계산** — 월세, 인건비, 객단가, 원가율, 배달 비중 입력 시 하루 필요 판매량 계산
- **폐업 이력 분석** — 희망 주소지 반경 내 유사 업종 폐업 현황 조회
- **부검 리포트** — 폐업 매장의 비용 구조, 리뷰, 경쟁 매장, 상권 환경 분석

## 기술 스택

- React 18
- Vite
- CSS (Pretendard 폰트)

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 대상 사용자

| 구분 | 대상 |
|------|------|
| B2C | 예비 창업자 |
| B2B | 세무사, 노무사, 창업 컨설턴트 |
| B2G | 소상공인 지원기관, 창업보육센터 |
